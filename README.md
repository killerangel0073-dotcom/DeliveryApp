# DeliveryApp

Proyecto de aplicación móvil desarrollado en **Android Studio** utilizando **Kotlin** y **Jetpack Compose**.

## Descripción general
Esta es una aplicación de ejemplo construida con Android moderno.  
El objetivo del proyecto es experimentar, practicar y estructurar una app usando tecnologías actuales de Android.

## Tecnologías utilizadas
- Kotlin  
- Jetpack Compose  
- Android Studio  
- Arquitectura moderna de Android  

## Estructura básica
El proyecto está organizado en diferentes paquetes para mantener el código ordenado, siguiendo buenas prácticas generales.

## Cómo ejecutar
1. Clonar este repositorio.  
2. Abrir el proyecto en Android Studio.  
3. Sincronizar Gradle.  
4. Ejecutar en un emulador o dispositivo físico.
